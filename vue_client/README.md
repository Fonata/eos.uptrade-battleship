## Transpilieren von TS zu JS
Damit Vue.js in der IDE hübsch aussieht, aber auch in alten Browsern funktioniert ist folgender Schritt nötig:

````bash
npm run build
````

Das erstellt eine Datei `api/public/js/vue_client.js`.
